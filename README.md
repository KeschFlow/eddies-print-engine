# Kids Activity Book Generator

Ein einfaches Python-Tool zur automatischen Erstellung von druckfertigen
PDF-Aktivitätsbüchern für Kinder (Malvorlagen, Punkt-zu-Punkt-Rätsel),
optimiert für Amazon KDP.

## Features
- Generiert vollständige PDF-Bücher
- Schwarz-weiß, ideal für Ausmalbücher
- Punkt-zu-Punkt-Seiten & einfache Lernaktivitäten
- Keine Texte nötig – kinderfreundlich & selbsterklärend
- KDP-taugliches Seitenlayout
